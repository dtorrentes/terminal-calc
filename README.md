# Terminal Calculator (Python CLI)

A simple command-line calculator built using **Replit** and published on **GitHub**.  
Allows users to perform basic arithmetic operations through a terminal interface.

## Features
- Supports addition, subtraction, multiplication, division, and exponentiation
- Input validation for numeric values
- Graceful handling of divide-by-zero errors
- Clean, user-friendly CLI output

## How to Run
```bash
python3 main.py
